# NayePankh Dashboard

## Project Overview

NayePankh dashboard is a Data Analytics project developed to analyze student participation, attendance, satisfaction, and dropout risks in NGO educational programs.

## Objectives

* Track student attendance and engagement
* Identify students at risk of dropping out
* Analyze program performance
* Support data-driven decision making
* Provide actionable insights for intervention

## Tools Used

* Power BI
* CSV Dataset
* Data Analytics

## Dashboard Features

* Total Students KPI
* Average Attendance KPI
* Average Satisfaction KPI
* Dropout Analysis
* Program-wise Participation
* Village-wise Participation
* Risk Category Analysis

## Key Insights

* Total Students: 1000
* Average Attendance: 60.55%
* Average Satisfaction: 5.45
* Dropout students identified through dashboard analytics
* Risk categories used for early intervention planning
